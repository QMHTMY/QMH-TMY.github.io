Python数据结构的各种操作

发表于 2020-05-28 | 分类于 编程语言
| 字数统计: 1.8k | 阅读时长 ≈ 8

Python实现的数据有(字符，数字，元组)[不可变]，(列表，字典，集合)[可变]。每一种数据的操作都不同，现做一番总结。

不可变数据

数字操作 解释 举例
num.real 返回数的实部 (2+3j).real -> 2
num.imag 返回数的虚部 (2+3j).real -> 3
num.bit_length 返回数的字节长度 a=6; a.bit_length() -> 3
num.to_bytes(L,byteorder) 返回数的字节数组 a=6; a.to_bytes(2,‘big’) -> b‘\x06\x00’
int.form_bytes(bts,byteorder) 返回由给定字节数组所表示的整数。 int.form_bytes(b‘\x00\x10’,‘big’) -> 16
num.as_integer_ratio() 返回一对整数，其比率为num a=6.0; a.as_integer_ratio() -> (6,1)
num.is_integer() 若num能用有限位整数表示返回True a=6.0; a.is_integer() -> True
num.hex() 以16进制返回浮点数 a=6.0; a.hex() -> ‘0x1.8000000000000p+2’
字符串操作 解释 举例(‘Ab ab’)
del sr 删除sr
sr[0] 通过下标取元素 ‘A’
st[1:3] 字符串切片 ‘b a’
sr[::-1] 字符串反转 ‘ba bA’
sr.lower() 转换为小写 ‘ab ab’
sr.upper() 转换为大写 ‘AB AB’
sr.title() 单词首字母大写 ‘Ab Ab’
sr.casefold() 转换为小写，对所有字符，包括法国字母 ‘ab ab’
sr.swapcase() 转换大写为小写，小写为大写 ‘aB AB’
sr.capitalize() 首字母大写 ‘Ab ab’
sr.center(10,‘-’) 将sr居中对齐并扩展到长为10的字符串，不够用-补齐 ‘–Ab ab—’
sr.ljust(10,‘-’) 将sr左对齐并扩展到长为n的字符串，不够用sp补齐 ‘Ab ab—–’
sr.rjust(10,‘-’) 将sr右对齐并扩展到长为n的字符串，不够用sp补齐 ‘—–Ab ab’
sr.zfill(10) 将sr右对齐并扩展到长为n的字符串，不够用0补齐 ‘00000Ab ab’
sr.split(‘ ’) 按‘ ’分隔字符串sr为列表，‘ ’默认为空格 [‘Ab’, ‘ab’]
sr.rsplit(‘a’,1) 按sp从右到左分隔字符串sr为列表，sp默认为空格，n为分割的次数 [‘Ab ’, ‘b’]
sr.startwith(‘a’) 判断sr是否以sp开头 False
sr.endswith(‘ab’) 判断sr是否以sp结尾 True
sr.expandtabs(4) 将sr中的tab键转换为空格，4为空格数 ‘A\tB’ -> ‘A B’
sr.isupper() 判断sr是否是大写 False
sr.islower() 判断sr是否是小写 False
sr.isalpha() 判断sr是否是字母 False
sr.isdigit() 判断sr是否是数字 False
sr.isalnum() 判断sr是否是字母数字 False
sr.isspace() 判断sr是否是空格 False
sr.istitle() 判断sr中单词首字母是否是大写 False
sr.isdecimal() 判断sr是否是十进制数 False
sr.isnumeric() 判断sr是否是数 False
sr.strip() 去掉sr两侧的空字符 ‘ Ab ab ’->‘Ab ab’
sr.lstrip() 去掉sr左侧的空字符 ‘ Ab ab ’->‘Ab ab ’
sr.rstrip(‘ ’) 去掉sr右侧的空字符 ‘ Ab ab ’->‘ Ab ab’
sr.encode(encoding) 按encoding规则编码sr
sr.decode(encoding) 按encoding规则解码sr
sr.count(‘a’) 统计sr中a的频度 2
sr.find(‘a’,s,e) 返回sr中从左侧开始第一个a位置，无则返回-1，可指定查找范围s和e 3
sr.rfind(‘a’) 返回sr中从右侧开始第一个a位置，无则返回-1，可指定查找范围s和e 3
sr.index(‘a’) 返回sr中从左侧开始第一个a位置，无则报错，可指定查找范围s和e 3
sr.rindex(‘a’) 返回sr中从右侧开始第一个a位置，无则报错，可指定查找范围s和e 3
sr.join(‘ABC’) 将sr作为ABC的连接符号，组成AsrBsrC这种字符串 ‘AAb abBAb abC’
sr.format() 格式化输出，替代%方法 ‘{}-{}’.format(1,2) -> ‘1-2’
sr.replace(‘a’,‘b’) 将sr中的a替换成b ‘Ab bb’
元组操作 解释 举例(1,2,3,4,1)
del tpl 删除tp
tpl[0] 索引元素 1
tpl[1:3] 切片 (2,3)
tp1+tp2 元组合并 (1,2) + (2,3) -> (1,2,2,3)
sorted(tpl) 按某规则排序tpl
tpl.count(‘1’) 统计tpl中1的频度 2
tpl.index(‘1’) 返回tpl中第一个x位置 0

可变数据

列表操作 解释 举例[1,2,3,4,1,2,3,4]
del lst[1] 删除列表lst中某元素 [1,3,4,1,2,3,4]
del lst 删除列表lst
‘-’.join(lst) 用-拼接lst为字符串 ‘1-3-4-1-2-3-4’
lst.append(‘x’) 向lst末尾添加元素x [1,3,4,1,2,3,4,‘x’]
lst.extend(iter) 将可迭代对象添加到lst lst.extend([10,9])->[1,3,4,1,2,3,4,10,9]
lst.insert(0,‘x’) 在lst的i位置插入x [‘x’,1,3,4,1,2,3,4]
lst.remove(1) 移除lst中的第一个1 [‘x’,3,4,1,2,3,4]
lst.pop(0) 弹出lst中2位置的元素 [3,4,1,2,3,4]
lst.index(1,start,end) 在start-end范围内查找元素，并返回位置 2
lst.count(3) 统计lst中1的频度 2
lst.sort(key=None, reverse=False) 对lst进行排序 [1,2,3,3,4,4]
lst.reverse() 对lst进行逆序 [4,4,3,3,2,1]
lst.clear() 清除lst中所有元素 []
lst.copy() 浅拷贝lst
copy.deepcopy(lst) 深拷贝lst
collections中的双队列 解释 举例deque([1,2,3,4])
copy(dq) 浅拷贝
append(5) 队列末尾添加元素5 deque([1,2,3,4,5])
appendleft(5) 队列首部添加元素5 deque([5,1,2,3,4,5])
pop() 弹出队列尾部数据 deque([5,1,2,3,4])
popleft() 弹出队列首部数据 deque([1,2,3,4])
extend([7,8,9]) 添加多个元素到队列尾部 deque([1,2,3,4,7,8,9])
extendleft([0,0,7]) 添加多个元素到队列首部 deque([0,0,7,1,2,3,4,7,8,9])
rotate(-2) 向左移动两个位置，负左，正右 deque([7,1,2,3,4,7,8,9,0,0])
remove(7) 删除第一个7 deque([1,2,3,4,7,8,9,0,0])
reverse() 队列倒序 deque([0,0,9,8,7,4,3,2,1])
字典操作 解释 举例{‘x’:1,‘z’:2,‘y’:3}
dct[‘x’] 索引元素x 1
del dct[‘x’] 删除元素x {z:2,y:3}
del dct 删除字典
dct.pop(‘x’) 弹出元素x {z:2,y:3}
dct.popitem() 弹出一个元素，一般为最后一个 {x:1,z:2}
dct.get(‘x’,‘s’) 获取元素x，若没有则返回s 1
dct.setdefault(‘m’,4) 获取元素m，若没有则设置m=4 {x:1,z:2,y:3,m:4}
dct.fromkeys(seq, val) 由seq和val创建字典
dct.update(dct1) 将dct1合并到dct中 dct1={‘a’:1} {‘x’:1,‘z’:2,‘y’:3,‘a’:1}
dct.keys() 获取dct的所有键 dict_keys([‘x’,‘z’,‘y’])
dct.values() 获取dct的所有值 dict_keys([1,2,3])
dct.items() 获取dct的所有键值 dict_keys([(‘x’,1),(‘z’,2),(‘y’,3)])
dct.copy() 浅拷贝dct
dct.clear() 清除dct中所有元素 {}
集合操作 解释 举例{1,2,3}
st.issubset(t) 判断st是否是t的子集合t={1,2} False
st.issuperset(t) 判断st是否是t的子集合 True
st.union(t) 集合并操作 st or t {1,2,3}
st.intersection(t) 集合交操作 st & t {1,2}
st.difference(t) 集合差操作 st - t {3}
st.symmetric_difference(t) 集合异或操作 st ^ t {3}
st.update(t) 合并t到st {1,2,3}
st.intersection_update(t) 合并st和t的交集到st {1,2}
st.difference_update(t) 合并st和t的差集到st {3}
st.symmetric_difference_update(t) 合并st和t的对称差集到st {}
st.add(‘x’) 向st中添加元素x {1,2,3,‘x’}
st.copy() 浅拷贝st
st.remove(‘x’) 移除st中x元素，不存在会报错 {1,2,3}
st.discard(‘x’) 丢弃st中x元素，不存在不报错 {1,2,3}
st.pop() 弹出最后一个元素 {1,2}
st.clear() 清除st中所有元素 {}
