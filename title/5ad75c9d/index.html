C/Python/Go/Rust/Dart控制原语

发表于 2020-11-08 | 分类于 编程语言 | 阅读次数:
| 字数统计: 1.1k | 阅读时长 ≈ 6

本文为五种语言对比学习的第八篇：流程控制原语。

流程控制原语，主要管理代码执行逻辑和路径，包括判断，跳转，循环，错误与异常。无论是面向对象还是面向过程，流程控制都必不可少。

C流程控制原语[无错误处理]

  • if else
  • switch case
  • ?:
  • for
  • while
  • do while
  • goto continue break

Python流程控制原语

  • if elif else
  • for in range
  • for else
  • while
  • while else
  • continue break pass
  • assert raise try expect else finally

Go流程控制原语

  • if else error
  • switch case default fallthrough
  • select case default
  • for
  • goto continue break

Rust流程控制原语

  • if else
  • for in
  • while
  • loop break
  • match Result<T,E> panic!

Dart流程控制原语

  • if else
  • switch case default
  • for
  • for in
  • while
  • do while
  • continue break
  • ?: ?? ??=
  • assert throw rethrow try on catch finally
