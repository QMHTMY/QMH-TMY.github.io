本文为七种语言对比学习的第八篇：流程控制原语。

流程控制原语，主要管理代码执行逻辑和路径，包括判断，跳转，循环，错误与异常。无论是面向对象还是面向过程，流程控制都必不可少。

C流程控制原语[无错误处理]

  • if else
  • switch case
  • ?:
  • for
  • while
  • do while
  • goto continue break

C++流程控制原语

  • if else
  • switch case
  • try catch throw
  • ?:
  • for
  • while
  • do while
  • goto continue break

Java流程控制原语

  • if else
  • for foreach do while
  • switch case
  • continue break

Python流程控制原语

  • if elif else
  • for in range
  • for else
  • while
  • while else
  • continue break pass

Go流程控制原语

  • if else
  • switch case default fallthrough
  • select case default
  • for
  • goto continue break

Rust流程控制原语

  • if else
  • for in
  • while
  • loop break

Dart流程控制原语

  • if else
  • switch case default
  • for
  • for in
  • while
  • do while
  • continue break
  • ?: ?? ??=
