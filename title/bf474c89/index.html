C、C++、Java、Python、Go、Rust、Dart常量和变量

发表于 2020-11-07 | 分类于 编程语言
| 字数统计: 1.5k | 阅读时长 ≈ 7

本文为七种语言对比学习的第六篇：常量和变量。

编程语言中的量是一种表述和存储模型，它表征了某个对象或事物在内存中的保存和解释方式。这些量即有值又有类型，同时还有保存地址。常量就是值和类型均不能变的量，相反，变量的值可以变，但类型不可变，除非重定义。

C常量和变量

C中常量

  • define 定义
  • const 定义

C中变量

  • type variable 定义方式1
  • type variable = value 定义方式2
  • type variable1, variable2 定义方式3
  • type variable1 = value1, variable2 = value2 定义方式4
  • extern type variable 定义方式5

C++常量和变量

C++中常量

  • define 定义
  • const 定义 (多用)

C++中变量

  • type variable 定义方式1
  • type variable = value 定义方式2
  • type variable {value} 定义方式3
  • type variable1, variable2 定义方式4
  • type variable1 = value1, variable2 = value2 定义方式5
  • extern type variable 定义方式6

Java常量和变量

Java中常量定义方式

  • final type varname = value;
  • final static type varname = value;
  • public final type varname = value;

Java中变量

  • type variable 定义方式1
  • type variable = value 定义方式2
  • type variable {value} 定义方式3
  • type variable1, variable2 定义方式4
  • type variable1 = value1, variable2 = value2 定义方式5
  • extern type variable 定义方式6

Python常量和变量

Python中并没有C/C++中那种命名常量，一切都看使用形式。数字、字符串、布尔值、空值等都是常量。

Go常量和变量

Go中基本变量包括布尔型，数字型，字符串，它的定义和C/C++一样使用const。

Go中常量

  • const 定义

Go中变量

  • var定义
  • := 定义

Rust常量和变量

Rust中常量定义和C中相似，只是Rust加入了生命周期概念，和C++得智能指针有些类似，但Rust得这个概念比C++强大。

Rust中常量

  • const 定义常量，类似C/C++中#define
  • static 定义全局变量，可当常量使用

Rust中变量

  • let 定义，值不能变，但变量能重定义
  • let mut 定义，值能变，变量也能重定义
  • static mut 定义，静态变量，但值可变

Dart常量和变量

Dart中常量

  • const 定义并赋值
  • final 定义时可不赋值，但只能赋值一次

final和const区别：final修饰的是变量，定义时可不赋值，但初始化赋值只能一次，之后就不再变，类似常量。而const一开始修饰的就必须是常量，且必须初始化。final用时才申请内存，const一开始就处理好了，所以速度要快些。

Dart中变量

  • var 定义
  • dynamic 定义，类型可变
  • type variable 定义
